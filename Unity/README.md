# Instructions
1. Set up ROS instance according to [these](https://github.com/Unity-Technologies/Unity-Robotics-Hub/blob/dev/tutorials/pick_and_place/0_ros_setup.md) instructions.
2. In ROS workspace, launch Part 3: roslaunch niryo_moveit part_3.launch
3. In Unity project, open "Scenes/RobotStream"
4. In Robotics -> Ros Settings, configure ROS and Unity IP Addresses
5. Download Web Server with  Edit -> Render Streaming -> Download web app, locate and Run.
6. Play the scene in the editor.
7. Open browser and navigate to http://localhost
8. "Light On" will start motion planning. Other buttons do nothing.
